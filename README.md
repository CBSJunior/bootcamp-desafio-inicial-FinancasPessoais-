# Sistema Sem Dúvida$$$
> **Bootcamp II - Desafio Inicial** > Aplicação para Acompanhar as Finanças Pessoais

O **Sistema Sem Dúvida$$$** é uma aplicação desenvolvida em Python para auxiliar o controle financeiro pessoal, focando na rastreabilidade e memória de gastos cotidianos.

---

## 📝 Descrição do Problema Real
Muitas pessoas, ao consultarem a fatura do cartão de crédito no final do mês, deparam-se com descritivos de transações genéricos ou códigos de estabelecimentos que não remetem diretamente ao consumo realizado. Essa falta de clareza gera insegurança financeira e dificuldade em identificar onde o orçamento está sendo comprometido.

## 💡 Proposta da Solução
A aplicação oferece uma interface simples e direta para o registro imediato do gasto. Ao anotar o detalhe da compra no momento em que ela ocorre, o usuário cria um histórico auxiliar que serve como "memória" para conferência posterior da fatura, garantindo que nenhum gasto passe despercebido ou sem explicação.

## 👥 Público-Alvo
Proprietários de cartões de crédito e pessoas que buscam uma ferramenta ágil para registro de despesas variáveis e controle de fluxo de caixa pessoal.

## 🚀 Funcionalidades Principais
* **Registro de Gastos:** Entrada de valor, categoria e descrição detalhada.
* **Consulta de Histórico:** Visualização dos registros para conferência com a fatura.
* **Interface Amigável:** Fluxo de uso simplificado para facilitar o registro no ato da compra.

---

## 🛠 Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Linter:** Flake8 / Pylint (para garantir a qualidade do código)
* **Testes:** Pytest / Unittest

## ⚙️ Instalação e Execução

### Pré-requisitos
Certifique-se de ter o Python instalado em sua máquina.

### Como instalar?
1. Clone o repositório:
   git clone [https://github.com/SEU_USUARIO/bootcamp-desafio-inicial-FinancasPessoais-.git](https://github.com/SEU_USUARIO/bootcamp-desafio-inicial-FinancasPessoais-.git)

2. Acesse a pasta do projeto:
   cd bootcamp-desafio-inicial-FinancasPessoais-

3. (Opcional) Crie um ambiente virtual:
   python -m venv venv
   source venv/bin/activate  # No Windows: venv\Scripts\activate


### Como usar?
Para iniciar a aplicação, execute o comando:
python main.py
